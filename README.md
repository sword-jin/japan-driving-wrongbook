# 日本驾考错题本

单文件 HTML 错题本，用于复习日本驾驶执照笔试（学科试験）中做错的题目。原题为日语、释义为简体中文。

## 使用

在浏览器中打开 [index.html](index.html) 即可。无需构建，无需依赖。

## 功能

- 按类别筛选（信号灯 · 标识 · 行车操作 · 行人道 · 驾照 · 手势信号 …）
- 做题模式：随机抽取错题重新作答，即时反馈
- 标记「已掌握」并跟踪剩余待复习数量
- 难度星级（1–3 ★）
- 交通标识通过内嵌 SVG 呈现，零外部资源

## 添加新错题

卡片模板、标签约定与截图工作流详见 [CLAUDE.md](CLAUDE.md)。

## 技术栈

- 单文件 `index.html`（HTML + CSS + vanilla JS）
- Google Fonts：Noto Sans SC · Noto Serif SC
